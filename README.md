This is a 2D Action/Adventure prototype made by Darcy and Fel in the IVGD program at Dawson College. The game takes 5-10 minutes to play. Download the release zip package and run the application titled "Prototype 2 - Hardboiled" to play. You may get a warning before playing, if you don't trust me, then don't play the game!

All image sprites and code are done by us from scratch. The music and sfx are sourced below. We hope you enjoy the game!

Assets: 
Music - https://tallbeard.itch.io/music-loop-bundle
SFX - https://ci.itch.io/400-sounds-pack, https://filmcow.itch.io/filmcow-sfx
