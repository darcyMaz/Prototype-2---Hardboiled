## "Gigi's Gone Missing... Again" is a 2D prototype game made in Unity. All image assets as well as the code were made by us. The music and sound effects are sourced at the bottom.

Gino (or Gigi as most people know him) has gone missing. He probably got caught up with the wrong people... again. The whole family's counting on you to find sweet little Gigi.
